ORACLE密码更改

2021-06-12
数据库
  • Oracle
有 N 人看过

1. 更改命令：

C:\Users\Administrator>sqlplus / as sysdba

SQL*Plus: Release 11.2.0.2.0 Production on 星期六 6月 12 21:12:54 2021

Copyright (c) 1982, 2014, Oracle.  All rights reserved.


连接到:
Oracle Database 11g Express Edition Release 11.2.0.2.0 - 64bit Production

SQL> ALTER USER SYS ACCOUNT UNLOCK;

用户已更改。

SQL> ALTER USER SYS IDENTIFIED BY "TEST@123";

用户已更改。

2. Premium连接：

在使用Premium连接时报错：ORA-28009: connection as SYS should be as SYSDBA or SYSOPER
需要在高级选项中角色选择SYSDBA。如果是命令行的话：connect sys/sys as sysdba。


Oracle 11g默认的服务名为ORCL,Oracle 9i默认的服务名为OEMREP。
